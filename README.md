# Cocktail-Card
Application to brew cocktails and read more about recipes and ingredients / glasses. Using private api as much I can from here. 

# TechStack: Jetpack Compose, Kotlin Coroutines, MVVM, Koin, Room, Retrofit, Coil, Material/Material3
